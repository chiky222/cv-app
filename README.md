## Pasos para levantar el proyecto en local
<br>

1.  **Clonar el repositorio**

```
git clone https://github.com/chiky222/Lista-de-tareas
```

2. **Instalar las dependencias**

```
npm install
```

3. **Correr el proyecto en el navegador**

```
npm start
```
